# Code of Conduct

This project follows the [GNOME Code of Conduct](https://conduct.gnome.org/).

In short:

- **Be friendly.** Use welcoming and inclusive language.
- **Be empathetic.** Be respectful of differing viewpoints and experiences.
- **Be respectful.** Disagreement is fine; disrespect is not.
- **Be considerate.** Remember that decisions affect other contributors and users.
- **Be patient and generous.** Especially with newcomers.
- **Try to be concise.** Respect people's time.

Unacceptable behavior — harassment, discrimination, personal attacks, or
sustained disruption — will not be tolerated.

To report a problem, please contact the maintainer by opening a confidential
issue or emailing the address on the maintainer's GitHub profile.
